# config

My personal configuration files (dotfiles) for keeping text editors, keybindings, and development environments aligned across devices. This repository is built around my love for Vim motions and features a highly customized, terminal-centric workflow.

## Key Features

*   **Vim & Neovim Optimized:** Rooted in native Vim design principles, extending powerful modal workflows to every piece of software I use.
*   **Unified Vim-Style Workflows:** Brings consistent leader-key sequences (`Space`) and instant insert mode escapes (`jj`, `jk`, `kk`) to modern GUI editors like Zed and VS Code.
*   **Enhanced List Navigation:** Customized quick-open menus and tab switchers to navigate vertically using custom shift bindings rather than relying on arrow keys.
*   **Automated Formatting:** Pre-configured settings to format source code seamlessly on save/paste using ecosystem-standard formatters.
